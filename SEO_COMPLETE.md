## 📝 Content Guidelines

### Title Tags
- Include primary keyword
- Keep under 60 characters
- Make it compelling and clickable
- Include brand name at the end
- Format: "Primary Keyword - Secondary Keyword | Brand"

### Meta Descriptions
- Include primary and secondary keywords
- Keep between 150-160 characters
- Include a call-to-action
- Make it unique for each page
- Highlight unique value proposition

### Headings
- One H1 per page (main title with primary keyword)
- Use H2 for main sections (include related keywords)
- Use H3 for subsections
- Include keywords naturally
- Make them descriptive and engaging

### Content
- Write for humans first, search engines second
- Use keywords naturally (2-3% density)
- Include related keywords and synonyms
- Aim for 1000+ words for main pages
- Use bullet points and lists
- Add images and videos
- Update content regularly
- Answer user questions
- Provide value and solutions
